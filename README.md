
# Analizador Sintactico

## Requisitos:

- Java JDK 8  
- Netbeans 8.2

## Como ejecutar el analizador sintactico:

- Abrir proyecto en netbeans 
- run app 

## Pruebas 

- Ingresa una cadena 
- clic en el boton Analizar
